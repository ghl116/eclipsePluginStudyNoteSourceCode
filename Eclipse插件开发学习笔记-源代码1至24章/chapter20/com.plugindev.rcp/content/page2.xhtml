Game Instruction

1.Game Description

Win the Game : frog return home with 30 seconds.

Lose the Game: frog crashed by car or track;eaten by snake;fail to jump on to log and drown in river.

Prolong the time: frog eat the mosquito can gain 10 seconds.

2.Control Instruction

Start Game: Double click the View.

Frog Up: Up Arrow Key.

Frog Down: Down Arrow Key.

Frog Left: Left Arrow Key.

Frog Down: Right Arrow Key.

Exit Game: Game->Exit
